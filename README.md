<h1>Racine cli</h1>

<p>
A CLI for generating starter files with TailwindCSS pre-installed
</p>
<h3>Installation</h3>

```bash
npm i @babsfullstack/racine -g
```
You can remove `-g` if you don't want to install it globally


<h3> How to use ?</h3>

use `tcli` to execute the package
happy coding !!
